El mesero es la API

El intermediario que conecta dos sistemas sin que uno tenga que conocer los detalles internos del otro.

Una app del clima llama a la API de un servicio meteorológico → obtiene los datos → los muestra. Sin satélites propios.

La solución: MCP

Antes del USB

Cada dispositivo tenía su propio conector. Un caos.

USB lo resolvió

Un estándar universal para todo.

MCP hace lo mismo

Para la IA. Un protocolo que convierte al modelo en un sistema capaz de actuar.

Arquitectura cliente-servidor

MCP actúa como un puente entre la IA y el mundo real.

→ Host

El Host es la cara visible.

→ Cliente

El Client habla el idioma de MCP.

→ Servidor

El Server expone las capacidades reales al modelo.

¿Qué puede ofrecer un servidor MCP?

Un servidor MCP expone tres primitivas:

🔧

Tools

Controlado por el modelo

Funciones ejecutables: buscar en internet, controlar navegador, crear archivos.

🗃️

Resources

Controlado por la app

Fuentes de datos: esquema de una BD, documentos, historial.

📄

Prompts

Controlado por el usuario

Plantillas reutilizables: flujos predefinidos, guías de estilo.

Tipos de extensibilidad

🔌

APIs externas

Servicios de clima, pagos, redes sociales, cualquier API pública o privada.

🗄️

Bases de datos

Consultas en tiempo real, lenguaje natural → SQL.

📁

Sistema de archivos

Leer, escribir, organizar documentos.

🌐

Navegadores web

Automatización, screenshots, testing.

Lo más poderoso: puedes combinarlos todos en una sola instrucción.

Servidores MCP destacados

El ecosistema ya existe. Aquí algunos imprescindibles.

🎭

PLAYWRIGHT MCP

Automatización de navegadores

Le da a Claude control real sobre un navegador: abrir páginas, hacer clic, llenar formularios, tomar screenshots — todo desde lenguaje natural. Es el servidor que veremos en la demo.

🎨

FIGMA MCP

Diseño → Código

Conecta Claude con tus archivos de Figma para extraer variables, componentes y estilos, y convertirlos en código que ya respeta tu sistema de diseño. También puede escribir directamente en el canvas de Figma. Disponible gratis en beta.

📚

CONTEXT7 MCP

Documentación siempre actualizada

Soluciona el mayor problema del vibe coding: la IA usando docs desactualizadas. Context7 inyecta la documentación oficial y actualizada de cualquier librería justo antes de que Claude responda. Más de 51K ⭐ en GitHub.
